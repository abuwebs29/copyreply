# CopyReply MVP

Production-ready Next.js starter for CopyReply.com with a searchable curated reply library.

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Upload this folder to a new GitHub repository.
2. Import that repository into Vercel.
3. Keep the default Next.js build settings and deploy.
4. In Vercel, add `copyreply.com` and `www.copyreply.com` under Domains.
5. In Namecheap Advanced DNS, add the exact DNS records Vercel displays.
6. Set one domain as the redirect to the other (recommended canonical: `copyreply.com`).

## Before AdSense application
Replace placeholder contact details, publish substantially more original pages, connect analytics and Search Console, test every page, and ensure Privacy/Terms accurately describe your setup.
