# CopyReply Growth Sprint 1

## Included
- Expanded the library from 40 to 96 reply topics
- Tone filtering on every reply page
- Visible FAQ section on every reply page
- FAQPage and BreadcrumbList structured data
- Personalization checklist and usage guide
- Six related-reply links per reply page
- Improved title, description, canonical, and Open Graph metadata
- Next.js 15.5.9 security patch line

## Deployment
Upload all files in this folder directly to the repository root. Keep Vercel configured to use `bun install` and `bun run build`.

The feature build was verified across 106 routes before the final security patch bump from Next.js 15.5.7 to 15.5.9.
