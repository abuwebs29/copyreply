# Sprint 6.3B — Export & Sharing

## Added

- Plain-text downloads from Reply Studio.
- Microsoft Word-compatible `.doc` downloads.
- Print-ready view with browser “Save as PDF” support.
- One-click handoff to the user's default email application.
- Native device sharing where supported.
- Fallback share-link copying on browsers without Web Share.
- Editable share links that reopen the selected Studio situation and customization settings.
- Privacy-friendly analytics events for exports, shares, and opened shared drafts.
- Google Analytics tag `G-SMP52V7Y3W` loaded globally using Next.js `Script`.

## Privacy

Reply drafts remain browser-based. Share links encode Studio settings in the URL and do not require a CopyReply account or server-side storage.
